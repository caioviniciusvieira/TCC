## Sobre o Projeto

MASTERPC é um projeto desenvolvido como Trabalho de Conclusão de Curso (TCC) para a ETEC Profa. Anna de Oliveira Ferraz, com foco em fornecer informações claras e acessíveis sobre componentes de computador, integrado a uma loja virtual que facilita a escolha e compra dos itens.

Este trabalho foi apresentado como exigência parcial para obtenção do título de **Técnico em Informática para Internet**.

**Aprovado em:** 28 de novembro de 2025

---

## Resumo

A dificuldade que muitas pessoas enfrentam ao tentar compreender, escolher e montar componentes de computador motivou o desenvolvimento deste projeto, voltado especialmente a usuários que necessitam de suporte confiável e acessível.

O objetivo geral foi criar um site que oferecesse informações organizadas sobre hardware, integrado a uma loja virtual para aquisição dos componentes, facilitando o processo de decisão e diminuindo erros comuns na montagem.

Inicialmente, realizou-se uma análise de referência dos sites **KaBuM!**, **Pichau** e **UserBenchmark** para identificar funcionalidades relevantes. A partir disso, foram incorporados ao projeto recursos como barra de pesquisa, apresentação detalhada dos produtos e sistemas de carrinho e compra.

Optou-se por simplificar especificações e resultados de benchmark, tornando-os acessíveis a usuários iniciantes. Como diferencial, todas as informações técnicas e de desempenho foram organizadas diretamente na página de cada produto, facilitando a comparação.

Uma funcionalidade planejada — o teste automático de compatibilidade entre componentes — não foi implementada, pois outras prioridades foram consideradas mais relevantes.

Os resultados demonstram que o site alcançou o propósito de oferecer suporte informativo e comercial de forma integrada, apresentando navegação simples, conteúdo acessível e recursos práticos.

Conclui-se que o objetivo foi atingido, e o projeto possui potencial para futuras expansões, como novas funcionalidades, melhorias de usabilidade e ferramentas de apoio à montagem de computadores.
